Portfolio Performance Analysis: Growth vs. ValueThis project provides a quantitative analysis of two fundamental investment styles—Growth and Value—based on historical index data from 1984 to 2019. Using statistical metrics, the study evaluates which strategy offers better risk-adjusted returns.## OverviewThe goal of this analysis is to move beyond simple return observations and incorporate risk assessment into the investment decision-making process. By calculating volatility and efficiency ratios, we can determine how much "reward" an investor receives for every unit of "risk" taken.## Key Metrics AnalyzedThe project focuses on three primary financial indicators:Mean Annual Return: The average yearly profit generated by the fund.Standard Deviation: The measure of historical volatility and investment risk.Sharpe Ratio: Calculated as $\frac{R_p - R_f}{\sigma_p}$, where $R_p$ is the portfolio return, $R_f$ is the risk-free rate, and $\sigma_p$ is the standard deviation.## Technologies UsedGoogle Sheets / Microsoft Excel: For data processing and statistical calculations.Financial Modeling: Implementation of risk-adjusted performance formulas.Data Visualization: Comparative charts for Growth and Value indices.## Summary of FindingsThe analysis reveals that while the Growth portfolio typically yields higher absolute returns, it also carries significantly higher volatility. However, when adjusted for risk using the Sharpe Ratio, the Growth style still demonstrates a slight edge in efficiency over the Value style for the analyzed period.
